◆ SYS_ErrorType_t

enum SYS_ErrorType_t

‪Used to specify the type of an error.

Enumerator
SYSERR_WARNING 

A warning leads to the output of the warning message to the correct debug interfaces, but will not stop the execution of the program.

SYSERR_ERROR 

A normal error has the same effect of a warning, but the message printed to the debug interfaces is labeled as an error.

SYSERR_FATAL_ERROR 

A fatal error leads to the same message of a normal error, but will halt the execution of the program.
